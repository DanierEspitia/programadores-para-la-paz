Al enviar una petición POST a http://localhost:3000/registro con el JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde correctamente con un objeto JSON que confirma la recepción de los datos:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto demuestra que Express está leyendo correctamente req.body.

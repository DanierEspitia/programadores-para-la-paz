Backlog priorizado

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del caso.

2. Como coordinador quiero recibir una notificación en Telegram cuando se registre una nueva incidencia para realizar seguimiento oportuno.

3. Como miembro del equipo quiero consultar los reportes registrados para hacer seguimiento a los casos.

4. Como administrador quiero actualizar el estado de una incidencia para informar el avance de su atención.

5. Como ciudadano quiero consultar el estado de mi reporte para conocer si ya fue revisado.

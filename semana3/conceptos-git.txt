Repositorio:
Un repositorio es un espacio donde se almacenan los archivos y cambios de un proyecto.
Permite organizar el trabajo y mantener un historial de modificaciones realizadas.

Commit:
Un commit es un registro de los cambios realizados en un proyecto.
Sirve para guardar avances y tener control de las modificaciones hechas.

Push:
El comando push permite enviar los cambios del repositorio local hacia GitHub.
Esto ayuda a mantener actualizado el proyecto en línea.
